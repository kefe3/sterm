**STerm**, Serix OS Amber ekosistemi için **Kapsül Serix Takımı** tarafından geliştirilmiş, hibrit paket yönetimi ve özel komut setlerine sahip gelişmiş bir terminal katmanıdır.

## 🔄 OTA (Over-The-Air) Güncelleme
Sistemi her zaman güncel tutmak için artık tek bir komut yeterli:
- Terminale `sterm-update` yazarak GitHub üzerindeki en son özellikleri anında çekebilirsiniz.

## ⌨️ Kapsül Özel Kısayolları
Serix OS kullanımını hızlandırmak için tasarlanan yeni nesil kısayollar:

| Komut | Karşılık | Fonksiyon |
| :--- | :--- | :--- |
| **`yd`** | `cd` | **Y**er **D**eğiştir (Dizinler arası geçiş) |
| **`pd`** | `cp` | **P**arça **D**epola (Dosya kopyalama) |
| ****`my`** | `nano` | **M**etin **Y**az (Dosya düzenleme) |
| ****`he`** | `mv` | **H**edefe **E**riştir (Dosya taşıma) |

## 🚀 Kurulum ve Çalıştırma

```bash
# Depoyu klonlayın
git clone [https://github.com/kefe3/sterm.git](https://github.com/kefe3/sterm.git)

# Klasöre giriş yapın
cd sterm

# Çalıştırma izni verin ve başlatın
chmod +x sterm.py
./sterm.py
