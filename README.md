# ⚡ STerm - Terminal Ekosistemi

**STerm**, terminal kullanımını kolaylaştıran ve profesyonel araçları tek bir çatı altında toplayan Python tabanlı bir yardımcıdır. **Kapsül Serix Takımı** tarafından geliştirilmiştir.

---

## 🚀 Temel Özellikler
* ☁️ **Canlı Güncelleme:** Kendini internet üzerinden otomatik yeniler.
* 🛠️ **Gelişmiş Araçlar:** Hava durumu, parola üretici ve dosya indirici.
* 📂 **Dosya Yönetimi:** Kolay klasör oluşturma, silme ve disk analizi.
* 🇹🇷 **Türkçe Arayüz:** Tamamen yerli ve anlaşılır komut seti.

---

## 🛠️ Kurulum ve Başlatma

```bash
# Projeyi indirin
git clone [https://github.com/kefe3/sterm.git](https://github.com/kefe3/sterm.git)
cd sterm

# Gerekli kütüphaneyi kurun
pip install requests

# Çalıştırın
python sterm.py
