# ⚡ STerm - Terminal Ecosystem (Titan-Pro)

![Version](https://img.shields.io/github/v/release/kefe3/sterm?color=red&style=for-the-badge)
![Ecosystem](https://img.shields.io/badge/Ecosystem-Serix_OS_Amber-orange?style=for-the-badge)

**STerm**, sıradan bir terminal değil; Linux sistemleri üzerinde çalışan bir "Zırh" (Exoskeleton) ve kontrol katmanıdır. Kapsül Serix diliyle sistemi yönetmenizi sağlar.

## 🛠️ Temel Komut Seti

### 📦 Kapsül Kısayolları (V1.1 Legacy)
| Komut | Fonksiyon |
| :--- | :--- |
| `yd` | Dizin Değiştir (cd) |
| `pd` | Parça Depola (cp -r) |
| `my` | Metin Yaz (nano) |
| `he` | Hedefe Eriştir (mv) |

### 🔑 Sistem & Yetki (V1.3 Titan)
| Komut | Fonksiyon |
| :--- | :--- |
| `serix-kök` | Tam Yetkili Root Erişimi (sudo su) |
| `kapat` | Bilgisayarı Kapat |
| `yeniden` | Sistemi Yeniden Başlat |
| `kilitle` | Ekranı Kilitle |
| `serix-get`| Akıllı Paket Yükleyici |

### 📊 Analiz & İzleme
- **`gezgin`**: Gelişmiş dosya listeleme.
- **`analiz`**: Derin donanım raporu.
- **`günlük`**: İşlem geçmişini görüntüleme.
- **`temizle`**: Terminali temizler ve logoyu basar.
