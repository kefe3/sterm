⚡ STerm - Terminal EcosystemSTerm, Serix OS Amber ekosistemi için Kapsül Serix Takımı tarafından sıfırdan tasarlanmış gelişmiş bir terminal katmanıdır. Sistem yönetimi, paket kontrolü ve grafik arayüz geçişlerini tek bir merkezden yöneterek kullanıcıya kusursuz bir hibrit deneyim sunar.🛠️ Teknik ÖzelliklerHybrid Core: Debian (Ubuntu) ve Arch tabanlı sistemlerde otomatik uyumluluk ve paket yöneticisi tespiti.Serix-Get: Paket yönetim süreçlerini optimize eden, hata payını minimize eden özel komut seti.Amber GUI Dashboard: Terminal üzerinden gui komutuyla tetiklenebilen, grafiksel kontrol paneli desteği.System Metrics: Donanım (RAM/Disk) ve ağ durumunu izleyen entegre system-check ve speed-test araçları.🚀 Kurulum ve KullanımSistemi klonlamak ve çalıştırmak için terminalinize aşağıdaki komutları girin:Bash# Depoyu klonlayın
git clone https://github.com/kefe3/sterm.git

# Klasöre giriş yapın
cd sterm

# Çalıştırma izni verin
chmod +x sterm.py

# Terminali başlatın
./sterm.py
